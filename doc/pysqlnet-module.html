Module pysqlnet

source code

PySQL daemon mode Principle is quite simple. You start pysql as a daemon on one machine with Oracle client and pysql installed, then, everyone with a simple telnet client (like putty or telnet.exe on windows and legacy telnet on netcat on Unix) can use pysql !

This is intended for machine without Oracle client or without python (sad people !).
