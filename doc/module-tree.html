Module Hierarchy

  • pysql: Pysql is an Oracle client for all those who suffer from sqlplus.
  • pysqlcolor
  • pysqlconf: This module defines the PysqlConf class that handles all pysql configuration stuff
  • pysqldb: Database related stuff: Oracle interface (PysqlDb) and backgound queries (BgQuery)
  • pysqlexception: This module defines all pysql exceptions
  • pysqlfunctions: This module defines all high level functions of pysql
  • pysqlgraphics: This module defines all high level graphical functions of pysql
  • pysqlhelpers: This module defines some common useful functions
  • pysqlio: This module contains the PysqlIO
  • pysqlnet: PySQL daemon mode Principle is quite simple.
  • pysqloraobjects: This module defines some classes to make easier Oracle object manipulation
  • pysqlqueries: SQL queries ordered by theme in dictionary
  • pysqlshell: User interaction with pysql
  • pysqlupdate: This module check for PySQL updates and help user to update PySQL online
